This resource provides a crafting and dismantling system powered by an exquisite NUI, featuring built-in leveling system and tool requirements to elevate your server's gameplay experience.

## Features
- **Crafting & Dismantling:** Craft and dismantle items with ease.
- **Multi-Station Support:** Set up workbenches anywhere you choose.
- **Multi-Recipe Support:** Configure multiple crafting and dismantling recipes, even for the same item.
- **Job Locking:** Restrict specific workbenches to designated jobs or ranks
- **XP & Leveling:** Unlock advanced recipes as players gain crafting experience.
- **Tools Requirements:** Define tool requirements for specific recipes.
- **Queue System:** Players can queue multiple items at once.
- **Vehicles Support:** Includes the option to craft vehicles directly into a player‚Äôs garage.

## Highlights
- **Framework Support:** Compatible with popular frameworks (ESX / QBCore / QBox / Ox) and inventories (ESX / qb-inventory / ox_inventory).
- **Customisable Colour:** NUI theme colour can be customise in the settings file to match your server.
- **Easy Configuration:** Al configurable items are managed in the `data` folder.
- **Exports Support:** Export function is available for server integration.
