## 安裝
本頁說明如何將插件安裝至 FiveM 伺服器。

您亦可參照各插件的安裝指南，執行該插件特有的額外步驟。

### 步驟
- 前往 [COSMOS 商店](https://cosmos.tebex.io/) 領取所需插件。
- 前往 [Cfx Portal 頁面](https://portal.cfx.re/)，導航至 Assets 頁面。
- 在 **Granted Assets** 標籤頁下，定位至該插件並點擊 **Download**。
- 將下載的 zip 檔案解壓縮至 `resource` 資料夾，並將資料夾重命名為該插件名稱。

::: warning 警告
伺服器必須使用與購買資插件時所用**相同**的 Cfx 帳戶所產生的金鑰。
:::
